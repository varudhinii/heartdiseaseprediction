# Heart Disease Prediction System

## PROBLEM STATEMENT
Early detection of heart disease is critical to reduce mortality rates. This project predicts the likelihood of heart disease using machine learning techniques.

## TECHNOLOGIES USED
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## DATASET USED
UCI Heart Disease Dataset

## APPROACH
- Data preprocessing and scaling
- Model training using Machine Learning algorithms
- Model evaluation
- Deployment using Streamlit
